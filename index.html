AI-Powered Security Solution

AI 기반 자동화 취약점 진단, PoliCore

복잡한 IT 인프라, 더 이상 수동적인 보안에 의존하지 마세요. PoliCore는 공공기관의 디지털 자산을 AI 기술로 안전하게 보호합니다.

PoliCore의 특장점

최첨단 AI 기술과 방대한 보안 데이터를 기반으로 공공기관의 IT 자산을 효과적으로 보호합니다.

AI 기반 스마트 진단 엔진

자체 개발한 ATC(Automated Threat Correlation) 엔진이 자산 등록부터 분석, 보고서 생성까지 전 과정을 자동화합니다.

완벽한 보안 컴플라이언스 대응

ISMS-P, K-RMF, CSAP 등 국내외 주요 보안 기준의 취약점 진단 요소를 템플릿으로 제공합니다.

유연한 하이브리드 진단

Agent, Agentless, 수동 진단 방식을 모두 지원하여 시스템에 최적화된 방법을 선택할 수 있습니다.

실시간 취약점 분석

실시간 취약점 분석·평가를 통해 위험평가의 객관적 지표를 제공합니다.

편의성 극대화 Web UI

직관적인 통합 대시보드로 취약점 현황을 한눈에 확인하고 실시간 보고서를 생성합니다.

K-RMF 규제 준수 자동화

국방표준준수평가(K-RMF) 프로세스를 완전 자동화하여 공공기관의 규제 준수 부담을 경감합니다.

PoliCore 취약점 진단 자동화 솔루션

충청사이버보안아카데미(CCSA)의 다년간의 보안 컨설팅 노하우와 AI 기술력이 집약된 취약점 진단 자동화 솔루션입니다.

자산 그룹/권한 관리

자산별, 운영 부서별 그룹핑과 관리자/사용자별 접근 권한을 세분화하여 관리합니다.

진단 템플릿 관리

취약점 진단 기준 항목을 커스터마이징하여 내부 보안 지침에 맞게 설정합니다.

다양한 진단 수행

자산별, 그룹별, IP 대역별 다양한 진단 대상을 선정하고 수행 중 취소도 가능합니다.

상세 결과 가이드

Web UI에서 보고서를 확인하고 Excel, Word 형태의 상세 결과 보고서를 제공합니다.

신속한 조치 관리

사전조치, 결과조치 기능을 제공하며 조치 담당자와 기한을 지정할 수 있습니다.

효율적인 결재 관리

결재선을 지정하고 결재 요청 리스트에 대한 효율적인 이력관리를 지원합니다.
